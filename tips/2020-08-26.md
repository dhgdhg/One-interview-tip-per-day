
#### 描述

```python
a = 0
l = [0, 1, 2]

c, a = l[a], a + 1
print (c, a)

a, c = a + 1, l[a]
print (c, a)
```

请写出打印内容

答案

```python
In [1]: a = 0

In [2]: l = [0, 1, 2]

In [3]: c, a = l[a], a + 1

In [4]: print (c, a)
0 1

In [5]: a, c = a + 1, l[a]

In [6]: print (c, a)
1 2
```

原理

```python
a = 0
l = [0, 1, 2]

# c, a = l[a], a + 1
t1 = l[a]  # 0
t2 = a + 1 # 1

c = t1     # 0
a = t2     # 1

# a, c = a + 1, l[a]
t1 = a + 1 # 2
t2 = l[a]  # 1

c = t2     # 1
a = t1     # 2
```
